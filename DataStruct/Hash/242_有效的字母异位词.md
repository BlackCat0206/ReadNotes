# 242_有效的字母异位词

---- Lee

## 一、题目

给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

示例 1: 输入: s = "anagram", t = "nagaram" 输出: true

示例 2: 输入: s = "rat", t = "car" 输出: false

**说明:** 你可以假设字符串只包含小写字母

`LeetCode`链接：[242.有效的字母异位](https://leetcode.cn/problems/valid-anagram/description/)

### 二、思路

* 输入的`string`类型均包含只包含小写字符。
* 最终目的是验证一个`string`中的各个字符出现的次数与第二个字符串相同。
* 可以采用Hash的逻辑，将字母映射为数组的value，之后遍历数组，看数组是否整体为0。

```cpp
class Solution {
public:
    bool isAnagram(string s, string t) {
        
        // s于t均只包含小写字符
        // 通过hash统计字符串中每个字母出现的次数
        int count[26] = { 0 };

        for (char &c1 : s)
        {
            count[c1 - 'a']++;
        }

        // 将第二个string对象的char字符进行对比
        for (char &c2 : t)
        {
            count[c2 - 'a']--;
        }

        // 遍历字符统计数组，若均为0，则证明是
        for (int i = 0; i < 26; ++i)
        {
            if (0 != count[i])
            {
                // 若有元素不为0则说明不是字母异位词，返回false
                return false;
            }
        }
        
        // 若数组的元素均为0，则说明是字母异位词
        return true;
    }
};
```

- 时间复杂度: O(n)
- 空间复杂度: O(1)
