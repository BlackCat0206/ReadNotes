# 128_最长连续序列

---- Lee

## 一、题目

给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 `O(n)` 的算法解决此问题。

**示例 1：**

```
输入：nums = [100,4,200,1,3,2]
输出：4
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。
```

**示例 2：**

```
输入：nums = [0,3,7,2,5,8,4,6,0,1]
输出：9
```

**示例 3：**

```
输入：nums = [1,0,1,2]
输出：3
```

**提示：**

- `0 <= nums.length <= 105`
- `-109 <= nums[i] <= 109`

[力扣题目链接](https://leetcode.cn/problems/longest-consecutive-sequence/description/?envType=study-plan-v2&envId=top-100-liked)

## 二、思路

* Hash容器选择：要求时间复杂度是`O(n)`。传入数组的大小趋近于n，故**每次查询尽可能快**，且要考虑到**数组中有重复元素，要去重**。故选择`unordered_set`实现快速查找。
* 将数组中元素填充至`set`容器中，实现去重。

```cpp
unordered_set<int> set;
for (const int& num : nums){
    set.insert(num);
}
```

* **遍历容器（已去重，避免超时问题），如果它是起始元素再处理**：很重要，减少了很多不必要的操作

```cpp
for (const int& num : set){
    if (set.find(num - 1) == set.end()){
        // ... ...
    }
}
```

* 找到起点后，不断的查询其后的元素，直至为空，记录并更新最大长度

```cpp
int currentNum = num;
currentLength = 1;

while (set.find(currentNum + 1) != set.end()){
    ++currentNum;
    ++currentLength;
}

maxLength = max(currentLgenth, maxLength);
```

## 三、代码

```cpp
class Solution {
public:
int longestConsecutive(vector<int>& nums) {
    unordered_set<int> set(nums.begin(), nums.end());

    int maxLength = 0;
    int currLenght = 0;
    int currentNum = 0;

    // 遍历每个数字，寻找连续序列的起点
    for (const int& num : set){
        // 仅当num是序列的起点时才进行处理
        if (!set.count(num - 1)){
            currentNum = num;
            currLenght = 1;

            // 继续查询，直至不连续
            while (set.count(currentNum + 1)){
                currentNum += 1;
                currLenght += 1;
            }

            maxLength = max(currLenght, maxLength);
        }
    }
    
    return maxLength;
}
};
```
