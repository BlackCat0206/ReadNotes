# 138_随机链表的复制

---- Lee

## 一、题目

给你一个长度为 `n` 的链表，每个节点包含一个额外增加的随机指针 `random` ，该指针可以指向链表中的任何节点或空节点。

构造这个链表的 **[深拷贝](https://baike.baidu.com/item/深拷贝/22785317?fr=aladdin)**。 深拷贝应该正好由 `n` 个 **全新** 节点组成，其中每个新节点的值都设为其对应的原节点的值。新节点的 `next` 指针和 `random` 指针也都应指向复制链表中的新节点，并使原链表和复制链表中的这些指针能够表示相同的链表状态。**复制链表中的指针都不应指向原链表中的节点** 。

例如，如果原链表中有 `X` 和 `Y` 两个节点，其中 `X.random --> Y` 。那么在复制链表中对应的两个节点 `x` 和 `y` ，同样有 `x.random --> y` 。

返回复制链表的头节点。

用一个由 `n` 个节点组成的链表来表示输入/输出中的链表。每个节点用一个 `[val, random_index]` 表示：

- `val`：一个表示 `Node.val` 的整数。
- `random_index`：随机指针指向的节点索引（范围从 `0` 到 `n-1`）；如果不指向任何节点，则为 `null` 。

你的代码 **只** 接受原链表的头节点 `head` 作为传入参数。

**示例 1：**

![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2020/01/09/e1.png)

```
输入：head = [[7,null],[13,0],[11,4],[10,2],[1,0]]
输出：[[7,null],[13,0],[11,4],[10,2],[1,0]]
```

**示例 2：**

![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2020/01/09/e2.png)

```
输入：head = [[1,1],[2,1]]
输出：[[1,1],[2,1]]
```

**示例 3：**

**![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2020/01/09/e3.png)**

```
输入：head = [[3,null],[3,0],[3,null]]
输出：[[3,null],[3,0],[3,null]]
```

**提示：**

- `0 <= n <= 1000`
- `-104 <= Node.val <= 104`
- `Node.random` 为 `null` 或指向链表中的节点。

## 二、思路

* 每日一遍：感谢灵神！
* 深拷贝，新构建每个节点，如果只是顺序链表，单纯拷贝即可。但涉及到了`random`就很难通过元素大小判断指向了。
* 依次复制每个节点（创建新节点并复制 *val* 和 *next*），把新节点直接插到原节点的后面，形成一个**交错链表**：

```cpp
// 源节点不为空
while (nullptr != cur){
    ListNode* node = new ListNode(cur->val);
    node->next = cur->next;
    cur->next = node;
    
    // 移动两个单位
    cur = cur->next->next;
}
```

* 遍历交错链表，若源节点`random`不为空。
* 将新节点的`random`（`cur->next->random`）指向源节点`random`的下一个对象`cur->random->next`。
* 每次移动两个节点。注意边界条件，移动的条件是当前节点后还有节点。

```cpp
// 注意边界条件，移动的条件是当前节点后还有节点。
while (nullptr != cur && nullptr != cur->next){
    if (nullptr != cur->random){
        cur->next->random = cur->random->next;
    }
    cur = cur->next->next;
}
```

* 分离两个链表

```cpp
ListNode* newHead = head->next;
ListNode* cur = head;
// 判断是否到最后一个节点
while (nullptr != cur->next && nullptr != cur->next->next){
    ListNode* copy = cur->next;
    cur->next = copy->next;
    copy->next = copy->next->next;
    // 位移
    cur = cur->next;
}
cur->next = nullptr;	// 最后一个节点的next至空。	
```

## 三、代码

```cpp
/*
// Definition for a Node.
class Node {
public:
    int val;
    Node* next;
    Node* random;
    
    Node(int _val) {
        val = _val;
        next = NULL;
        random = NULL;
    }
};
*/

class Solution {
public:
    Node* copyRandomList(Node* head) {
        // 入参校验
        if (nullptr == head){
            return head;
        }

        // 将新节点插入至源链表节点之后
        Node* cur = head;
        while (nullptr != cur){
            Node* node = new Node(cur->val);
            node->next = cur->next;
            cur->next = node;

            // 位移
            cur = cur->next->next;
        }


        // 遍历每一个源节点，新节点的random指向源节点random的下一个节点。
        cur = head;
        while (nullptr != cur && nullptr != cur->next)
        {
            // 判断random是否为空
            if (nullptr != cur->random){
                cur->next->random = cur->random->next;
            }
            // 位移
            cur = cur->next->next;
        }

        // 拆分链表
        Node* new_head = head->next;
        cur = head;
        // 判断是否到最后一个源节点
        while (nullptr != cur && nullptr != cur->next && nullptr != cur->next->next)
        {
            Node* copy = cur->next;
            cur->next = copy->next;
            copy->next = copy->next->next;
            // 位移一位
            cur = cur->next;
        }
        cur->next = nullptr;       // 最后一个节点元素next至空

        return new_head;
    }
};
```
