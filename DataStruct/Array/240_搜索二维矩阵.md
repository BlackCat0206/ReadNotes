## 一、题目

编写一个高效的算法来搜索 `*m* x *n*` 矩阵 `matrix` 中的一个目标值 `target` 。该矩阵具有以下特性：

- 每行的元素从左到右升序排列。
- 每列的元素从上到下升序排列。

**示例 1：**

![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2020/11/25/searchgrid2.jpg)

```
输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 5
输出：true
```

**示例 2：**

![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2020/11/25/searchgrid.jpg)

```
输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 20
输出：false
```

**提示：**

- `m == matrix.length`
- `n == matrix[i].length`
- `1 <= n, m <= 300`
- `-109 <= matrix[i][j] <= 109`
- 每行的所有元素从左到右升序排列
- 每列的所有元素从上到下升序排列
- `-109 <= target <= 109`

## 二、思路

* 选择大于努力：感谢灵神~\(≧▽≦)/~
* 左下角起点（官方解）：Z字执行，比当目标元素大向上走，比当前元素小向右走。注意：`x`和`y`的位置哦

```cpp
        // 从左下角开始遍历，增加向右走，减少向上走
        int x = 0;
        int y = matrix.size() - 1;
        while (x <= matrix[0].size() - 1 && y >= 0)
        {
            if (target == matrix[y][x])
            {
                return true;
            }
            else if (target > matrix[y][x])
            {
                x++;
            }
            else
            {
                y--;
            }
        }
```

* 右上角起点（灵神解）：排除法
  * 目标元素 < 当前元素：当前元素所在列排除，向左走。
  * 目标元素 > 当前元素：当前元素所在行排除，向下走。
