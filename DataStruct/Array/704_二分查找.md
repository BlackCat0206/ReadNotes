## 二、思路

* 有序数组，可以使用双指针控制左右边界。
* 左右边界如何设置？
  * 左闭右开：
    * [left, right) ,既`left = 0; right = sizeSums;` 
    * while循环的条件是`left < rigth` 
    * 当 `nums[middle] > traget`时修改右侧开区间；既`right = middle` 新的边界是`[lelft, middle)`
  * 左闭右闭：(这样写比较符合直觉)
    * [left, right], 既`left = 0; right = sizeSums - 1;`
    * `while`循环的条件是`left <= right`
    * 当`nums[middle] > target`修改右侧闭区间；`right = middle - 1;`新的边界是`[left, middle - 1]`

## 三、代码

```c
int search(int* nums, int numsSize, int target) {
    
    // 入参校验
    if (NULL == nums)
    {
        return -1;
    }
    
    int left = 0;
    int right = numsSize - 1;
    int middle = 0;
    while (left <= right)   // 定义数组域为闭区间[left, right]
    {
        middle = left + (right - left) / 2;
        if (nums[middle] < target)
        {
            left = middle + 1;      // 设置新的搜索域[middle+1, right]
        }
        else if (nums[middle] > target)
        {
            right = middle - 1;     // 设置新的搜索域[left, middle-1]
        }
        else
        {
            return middle;
        }
    }

    return -1;  // 查找错误
}
```
