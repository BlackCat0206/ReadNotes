# 53_最大子数组和

---- Lee

## 一、题目

给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**子数组**是数组中的一个连续部分。

**示例 1：**

```
输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
输出：6
解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。
```

**示例 2：**

```
输入：nums = [1]
输出：1
```

**示例 3：**

```
输入：nums = [5,4,-1,7,8]
输出：23
```

**提示：**

- `1 <= nums.length <= 105`
- `-104 <= nums[i] <= 104`

## 二、思路

* [560_和为K的子数组.md](../DoublePointers/560_和为K的子数组.md)

* 思路相似，计算各元素的前缀和，最大前缀和减去最小前缀和就是结果。
* 注意：其中会有元素为负数，若按照历史思路会引入0，导致类似`[-1]`计算错误。故需要实时计算并更新结果。

## 三、代码

```cpp
int maxSubArray(vector<int>& nums){
    if (nums.emtpy()){
        return 0;
    }
    
    // 定义结果
    int ans = INT_MIN;	// 需要每次获取最大值，不能设置为0；
    int pre_sum = 0;		// 当前前缀和
    int min_pre_sum = 0;	// 当前最小前缀和 考虑到整个数组中只有一个正整数的情况。
    
    for (int i = 0; i < nums.size(); ++i){
        pre_sum += nums[i];
        ans = max(ans, pre_sum - min_pre_sum);	// 获取最大前缀之差
        min_pre_sum = min(pre_sum, min_pre_sum);	
    }
}
```
