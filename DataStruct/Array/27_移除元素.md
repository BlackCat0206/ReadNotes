# 27_移除元素

---- Lee

## 一、题目

给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并**原地**修改输入数组。

元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

示例 1: 给定 nums = [3,2,2,3], val = 3, 函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。 你不需要考虑数组中超出新长度后面的元素。

示例 2: 给定 nums = [0,1,2,2,3,0,4,2], val = 2, 函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。

**你不需要考虑数组中超出新长度后面的元素**

[力扣题目链接(opens new window)](https://leetcode.cn/problems/remove-element/)

## 二、思路

* 快慢指针
  * 遍历一次数组，
    * 若`fast`下标的元素不是target，将`fast`下标的元素赋值给`slow`下标的元素。`fast`和`slow`同时位移。
    * 若`fast`下标的元素是`target`，`fast`位移，继续遍历。
    * 返回新数组中的长度`slow`。

## 三、代码

```c
int removeElement(int* nums, int numsSize, int val) {
    // 入参校验
    if (NULL == nums || 0 >= numsSize)
    {
        return 0;
    }

    // 设置快慢指针
    int fast = 0;
    int slow = 0;

    // 遍历nums
    for (fast = 0; fast < numsSize; ++fast)
    {
        // 若当前值等于val, fast指针位移，slow不动
        if (val == nums[fast])
        {
            continue;
        }
        // 若当前值不等于，将fast下标中的元素赋值给slow下标中的
        nums[slow++] = nums[fast];
    }
    // 返回数组长度slow
    return slow;
}
```
