# 438_找到字符串中所有字母异位词

---- Lee

## 一、题目

给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的 **异位词** 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

**示例 1:**

```
输入: s = "cbaebabacd", p = "abc"
输出: [0,6]
解释:
起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。
```

 **示例 2:**

```
输入: s = "abab", p = "ab"
输出: [0,1,2]
解释:
起始索引等于 0 的子串是 "ab", 它是 "ab" 的异位词。
起始索引等于 1 的子串是 "ba", 它是 "ab" 的异位词。
起始索引等于 2 的子串是 "ab", 它是 "ab" 的异位词。
```

**提示:**

- `1 <= s.length, p.length <= 3 * 104`
- `s` 和 `p` 仅包含小写字母

[力扣题目链接](https://leetcode.cn/problems/find-all-anagrams-in-a-string/description/?envType=study-plan-v2&envId=top-100-liked)

## 二、思路

* 核心方法：**滑动窗口 + 字符计数**：通过滑动窗口维护子串`s`的字符计数，结合**固定长度字符数组统计**`p`的字符分布，判断窗口内子串是否为`p`的异位词。
* 字符计数：用长度26的数组统计`p`子串中各字符的出现次数，代表窗口匹配的字符数量。

```cpp
// 计算子串的字符计数
int charCount[26] = { 0 };
for (const char& c : p){
    ++charCount[c - 'a'];
}
```

* 窗口滑动：移动右指针`right`遍历，对遍历元素对应字符计数进行消减。若计数为负（**字符超量/完全不匹配**），逐步收缩左边界，直至当前窗口（`count[s[right] - 'a']`）计数合法（ `>= 0`）。
  * 字符超量：`s = "aab"，p = "ab"`; 在匹配到第二个字符`a`时，字符`a`的计数为负数，`left`右移，移动至下标`1`，恢复计数合法。
  * 完全不匹配：`s = abceabcd, p = "abcd"`，在匹配至字符`e`时，`left`会逐步收缩至`right+1`

```cpp
int left = 0;
for (int right = 0; right < s.size(); ++right){
    int c = s[right] - 'a';
    --charCount[c];
    
    while (charCount[c] < 0){
        ++charCount[s[left]];
        ++left;
    }
}
```

* 匹配验证：每次匹配后根据窗口的**左右边界元素个数**与**子串的长度**比较判断是否完全匹配。

```cpp
if (right - left + 1 == p.size()){
    res.push(left);
}
```

## 三、代码

```cpp
class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        // 入参校验
        if (s.empty() || p.empty() || p.size() > s.size()){
            return {};
        }

        // 定义返回结果
        vector<int> res;

        // 计算子串的字符计数
        int charCount[26] = { 0 };
        for (const char& c : p){
            ++charCount[c - 'a'];
        }
        // 记录子串的长度
        int sizeMax = s.size();
        int sizeMin = p.size();

        // 使用滑动窗口
        int left = 0;
        // 滑动窗口移动
        for (int right = 0; right < sizeMax; ++right){
            int c = s[right] - 'a';
            charCount[c]--;
            // 若出现不匹配，则之前的匹配均无效，恢复之前匹配的计数，逐步收缩left，
            while (charCount[c] < 0){
                ++charCount[s[left] - 'a'];
                left++;
            }

            // 每次匹配后计算当前匹配长度是否到达子串的长度。
            if (right - left + 1 == sizeMin){
                res.push_back(left);
            }
        }
        return res;
    }
};
```
