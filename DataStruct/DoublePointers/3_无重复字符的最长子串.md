# 3_无重复字符的最长子串

---- Lee

## 一、题目

给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长 子串** 的长度。

**示例 1:**

```
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。
```

**示例 2:**

```
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
```

**示例 3:**

```
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
```

**提示：**

- `0 <= s.length <= 5 * 104`
- `s` 由英文字母、数字、符号和空格组成

## 二、思路

**滑动窗口**：

* 使用数组统计每个元素出现的次数`ASCII`共128位。
* 若元素未出现，添加计数，滑动窗口右移，计算最大长度。
* 若元素出现，循环收缩窗口，直至去除重复元素。
* 使用`while`循环控制`start`与`end`。

## 三、代码

```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        if (s.size() <= 1){
            return s.size();
        }

        int maxLen = 0;
        int start = 0;
        int end = 0;
        int cnt[128] = { 0 };       // 统计元素出现次数

        while (end < s.size()){
            if (0 == cnt[s[end]]){
                cnt[s[end]]++;
                maxLen = max(maxLen, end - start + 1);
                end++;
                continue;
            }

            while (start < end && cnt[s[end]] != 0){
                cnt[s[start]]--;
                start++;
            }
        }
        return maxLen;
        
    }
};
```
