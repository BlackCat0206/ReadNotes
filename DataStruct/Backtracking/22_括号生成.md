# 22_括号生成

---- Lee

## 一、题目

数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。

**示例 1：**

```
输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]
```

**示例 2：**

```
输入：n = 1
输出：["()"] 
```

**提示：**

- `1 <= n <= 8`

## 二、思路

**决策树：**

* 每次只能两个选择：加左括号或右括号。
* 但不是左右决策都是正确的，需要裁剪：
  * 左括号数量不能超过`n`。
  * 右括号的数量不能超过当前左括号数量。否则会出现`())`这种无效组合。
* 决策树叶子节点：当左、右括号数量等于`n`时，就是一个有效结果。

## 三、代码

```cpp
class Solution {
    vector<string> ans;
    int m_n;
public:
    void backtracking(string& str, int left, int right){
        if (m_n == left && m_n == right){
            ans.push_back(str);
            return;
        }

        // 填充左侧括号
        if (left < m_n){
            str.push_back('(');
            backtracking(str, left + 1, right);
            str.pop_back();
        }

        // 右侧括号数量小于左侧括号
        if (right < left){
            str.push_back(')');
            backtracking(str, left, right + 1);
            str.pop_back();
        }
    }

    vector<string> generateParenthesis(int n) {
        ans.clear();
        m_n = n;
        string str;
        backtracking(str, 0, 0);
        return ans;

    }
};
```
