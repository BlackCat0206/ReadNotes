# 17_电话号码的字母组合

---- Lee

## 一、题目

给定一个仅包含数字 `2-9` 的字符串，返回所有它能表示的字母组合。答案可以按 **任意顺序** 返回。

给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

![img](https://pic.leetcode.cn/1752723054-mfIHZs-image.png)

**示例 1：**

```
输入：digits = "23"
输出：["ad","ae","af","bd","be","bf","cd","ce","cf"]
```

**示例 2：**

```
输入：digits = "2"
输出：["a","b","c"]
```

**提示：**

- `1 <= digits.length <= 4`
- `digits[i]` 是范围 `['2', '9']` 的一个数字。

## 二、思路

**数字和字母的映射：**

```cpp
const string letterMap[10] = {
    "", // 0
    "", // 1
    "abc", // 2
    "def", // 3
    "ghi", // 4
    "jkl", // 5
    "mno", // 6
    "pqrs", // 7
    "tuv", // 8
    "wxyz", // 9
};
```

如图所示：

![17. 电话号码的字母组合](https://file1.kamacoder.com/i/algo/20201123200304469.png)

图中可以看出**遍历的深度**，就是输入"23"的长度，而**叶子节点就是我们要收集的结果**，输出["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"]。

要使用一个index变量：表示遍历第几个数字了，也表示深度。

**回溯三部曲：**

* 确认回溯函数参数

```cpp
vector<string> result;
string s;
void backtracking(const string& digits, int index);
```

* 确认终止条件

```cpp
if (index == digits.size())
{
    result.push_back(s);
    return;
}
```

* 单层遍历逻辑

```cpp
int digit = digit[index] - '0';
string letters = letterMap[digit];
for (int i = 0; i < letter.size(); ++i)
{
    s.push_back(letter[i]);
    backtracking(digits, index + 1);
    s.pop_back();
}
```

## 三、代码

```cpp
class Solution {
public:
    const string letterMap[10] = {
        "", // 0
        "", // 1
        "abc", // 2
        "def", // 3
        "ghi", // 4
        "jkl", // 5
        "mno", // 6
        "pqrs", // 7
        "tuv", // 8
        "wxyz", // 9
    };
    vector<string> result;
    string s;
    void backstracking(string& digits, int index){
        if (index == digits.size()){
            result.push_back(s);
            return;
        }

        int digit = digits[index] - '0';
        string letters = letterMap[digit];
        for(int i = 0; i < letters.size(); ++i){
            s.push_back(letters[i]);
            backstracking(digits, index + 1);
            s.pop_back();
        }
        return;
    }

    vector<string> letterCombinations(string digits) {
        result.clear();
        s.clear();
        if (digits.size() == 0){
            return result;
        }

        backstracking(digits, 0);
        return result;
    }
};
```
