### 1.1 递归三要素：

写递归三要素如下，或许能有帮助：

* **确认递归函数的参数和返回值**：
  * 明确那些参数是要递归中处理的，在递归函数上加上这类参数。
  * 明确每次递归函数要返回什么，进而明确函数的返回值类型。
* **确认终止条件**：递归最常见的文件就是运行时栈溢出，这里就是没有确认好终止条件。
* **确认单层递归的逻辑**：
  * 明确每一层递归要处理的信息。在这里要重复调用来实现递归的逻辑。

### 1.2 示例：

* 明确递归函数的参数和返回值：在递归的过程中需要传入`vector`，C语言传入数组和其当前长度的指针。也不需要返回什么数据，所以递归的返回值类型就是：

  * `CPP`:

  ```cpp
  void traversal(TreeNode* node, vector<int> result);
  ```

  * `C`:

  ```c
  void traversal(struct TreeNode* node, int* result, int* resultSize)
  ```

* 确认终止条件：对于当前遍历，就是当前节点为空，就直接`return`:

  * `CPP`:

  ```cpp
  if (NULL == cur) return;
  ```

  * `C`：

  ```c
  if (NULL == cur) return;
  ```

* 确认单层递归的逻辑：前序遍历是中左右，所以在单层递归中，要先取中间节点的数值。

  * `CPP`：

  ```cpp
  vec.push_back(cur->val);	// 中
  traversal(cur->left, vec);	// 左
  traversal(cur->right, vec);	// 右
  ```

  * `C`：

  ```c
  result[(*resultSize)++] = cur->val;			// 中
  traversal(cur->left, result, resultSize);	// 左
  traversal(cur->rigth, result, resultSize);	// 右
  ```

### 3.1 前序遍历：

```cpp
void traversal(TreeNode* cur, vector<int>& res)
{
    if (NULL == cur) return;
	res.push_back(cur->val);		// 中
    traversal(cur->left, res);		// 左
    traversal(cur->right, res);		// 右
}
```

## 三、线索遍历

```cpp
class Solution {
public:
    vector<int> inorderTraversal(TreeNode* root) {
        vector<int> ans;

        while (root) {
            if (root->left) {
                // 找 root 的前驱 pre：在中序遍历中，root 的上一个节点
                // 从 root->left 开始，一直向右走，直到走到尽头，或者遇到指向 root 的线索（回到 root 的路）
                TreeNode* pre = root->left;
                while (pre->right && pre->right != root) {
                    pre = pre->right;
                }

                // root 的左子树尚未访问
                if (pre->right == nullptr) {
                    pre->right = root; // 建立线索（回到 root 的路），相当于把 pre->right 当作栈
                    root = root->left; // 访问左子树
                    continue;
                }

                // root 的左子树访问完毕，去掉线索，恢复原样
                pre->right = nullptr; // 注：如果调用完 inorderTraversal 不再使用这棵二叉树，这行代码可以去掉
            }

            // root 的左子树访问完毕
            ans.push_back(root->val); // 记录当前节点的值
            root = root->right; // 如果有右子树就访问右子树，没有就顺着线索回到指向的节点
        }

        return ans;
    }
};
```
