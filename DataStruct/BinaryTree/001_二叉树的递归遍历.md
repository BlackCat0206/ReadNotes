### 1.1 递归三要素：

写递归三要素如下，或许能有帮助：

* **确认递归函数的参数和返回值**：
  * 明确那些参数是要递归中处理的，在递归函数上加上这类参数。
  * 明确每次递归函数要返回什么，进而明确函数的返回值类型。
* **确认终止条件**：递归最常见的文件就是运行时栈溢出，这里就是没有确认好终止条件。
* **确认单层递归的逻辑**：
  * 明确每一层递归要处理的信息。在这里要重复调用来实现递归的逻辑。

### 1.2 示例：

* 明确递归函数的参数和返回值：在递归的过程中需要传入`vector`，C语言传入数组和其当前长度的指针。也不需要返回什么数据，所以递归的返回值类型就是：

  * `CPP`:

  ```cpp
  void traversal(TreeNode* node, vector<int> result);
  ```

  * `C`:

  ```c
  void traversal(struct TreeNode* node, int* result, int* resultSize)
  ```

* 确认终止条件：对于当前遍历，就是当前节点为空，就直接`return`:

  * `CPP`:

  ```cpp
  if (NULL == cur) return;
  ```

  * `C`：

  ```c
  if (NULL == cur) return;
  ```

* 确认单层递归的逻辑：前序遍历是中左右，所以在单层递归中，要先取中间节点的数值。

  * `CPP`：

  ```cpp
  vec.push_back(cur->val);	// 中
  traversal(cur->left, vec);	// 左
  traversal(cur->right, vec);	// 右
  ```

  * `C`：

  ```c
  result[(*resultSize)++] = cur->val;			// 中
  traversal(cur->left, result, resultSize);	// 左
  traversal(cur->rigth, result, resultSize);	// 右
  ```

## 三、前序、中序、后序遍历迭代实现

### 3.1 前序遍历：

* `CPP`：

```cpp
void traversal(TreeNode* cur, vector<int>& res)
{
    if (NULL == cur) return;
	res.push_back(cur->val);		// 中
    traversal(cur->left, res);		// 左
    traversal(cur->right, res);		// 右
}
```

* C:

```c
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     struct TreeNode *left;
 *     struct TreeNode *right;
 * };
 */
/**
 * Note: The returned array must be malloced, assume caller calls free().
 */

void traversal(struct TreeNode* cur, int* num, int* returnSize)
{
    if (NULL == cur)
    {
        return;
    }
    num[(*returnSize)++] = cur->val;        // 中
    traversal(cur->left, num, returnSize);  // 左
    traversal(cur->right, num, returnSize); // 右
    return;
}

int* preorderTraversal(struct TreeNode* root, int* returnSize) 
{
    int* nums = (int*)malloc(sizeof(int)*100);
    if (NULL == nums)
    {
        return NULL;
    }

    *returnSize = 0;
    traversal(root, nums, returnSize);

    return nums;
}
```

### 3.2 中序遍历

* `CPP`

```CPP
void traversal(TreeNode* cur, vector<int>& res)
{
    if (NULL == cur) return;
    traversal(cur->left, res);
    res.push_back(cur->val);
    traversal(cur->right, res);
}
```

* `C`：

```c
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     struct TreeNode *left;
 *     struct TreeNode *right;
 * };
 */

void traversal(struct TreeNode* cur, int* nums, int* returnSize)
{
    if (NULL == cur)
    {
        return;
    }

    traversal(cur->left, nums, returnSize);         // 左
    nums[(*returnSize)++] = cur->val;               // 中
    traversal(cur->right, nums, returnSize);        // 右
    return;
}

/**
 * Note: The returned array must be malloced, assume caller calls free().
 */
int* inorderTraversal(struct TreeNode* root, int* returnSize) {
    int* nums = (int*)malloc(sizeof(int) * 100);
    if (NULL == nums)
    {
        return NULL;
    }

    *returnSize = 0;
    traversal(root, nums, returnSize);
    return nums;
}
```
