## 一、二叉树的种类

### 1.1 二叉树

满二叉树：如果一棵二叉树只有度为0的结点和度为2的结点，并且度为0的结点在同一层上，则这棵二叉树为满二叉树。

如图所示：

![img](https://file1.kamacoder.com/i/algo/20200806185805576.png)

这棵二叉树为满二叉树，也可以说深度为k，有2^k-1个节点的二叉树。

### 1.2 完全二叉树

在完全二叉树中，除了最底层的节点，其余每一层的节点数都达到了最大值。且最下面一层的节点都集中在该层**最左边的若干位置**。若最底层为第h层，则该层包含**1~2^(h - 1)**。

![img](https://file1.kamacoder.com/i/algo/20200920221638903.png)

### 1.3 二叉搜索树

二叉搜索树是一颗有序树。

* 若它左子树不空，则左子树上所有节点的值都**小于**它的根节点的值。
* 若它右子树不空，则右子树上所有节点的值都**大于**它的根节点的值。
* 它的左、右子树分别为**二叉排序树**

![img](https://file1.kamacoder.com/i/algo/20200806190304693.png)

### 1.4 平衡二叉搜索树

`AVL (adelson-Velsky and Landis)`树，其性质为：它是一颗空树，或其左右子树的高度差的绝对值不超过1，且左右两棵子树也是平衡二叉搜索树。

![img](https://file1.kamacoder.com/i/algo/20200806190511967.png)

最后一棵 不是平衡二叉树，因为它的左右两个子树的高度差的绝对值超过了1。

C++中`set`,`multiset`,`map`，`multimap`底层实现都是平衡二叉搜索树，所有其对应的增删的时间操作复杂度是`logn`。

## 三、二叉树的遍历方式

### 3.1 主要的遍历方式

* 深度优先遍历：先往深走，遇到叶子节点再往回走。
  * 前序遍历 中左右（递归法，迭代法）
  * 中序遍历 左中右（递归法，迭代法）
  * 后序遍历 左右中（递归法，迭代法）
* 广度优先遍历：一层一层的去遍历。
  * 层序遍历（迭代法）

上述深度优先遍历提到的**前序，中序，后序**相关的内容，其均表示**中间节点的位置**。

![img](https://file1.kamacoder.com/i/algo/20200806191109896.png)

### 3.2. 遍历的实现方式

* 深度优先遍历：递归方式。之前说栈与队列时，栈其实就是递归的一种结构。
* 广度优先遍历：队列来实现的。

## 四、二叉树的定义

C++代码：

```cpp
struct TreeNode
{
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x): val(x), left(nullptr), right(nullptr) {}
};
```

C代码:

```c
struct TreeNode
{
    int val;
    TreeNode* left;
    TreeNode* right;
};

typedef TreeNode TreeNode;
```
