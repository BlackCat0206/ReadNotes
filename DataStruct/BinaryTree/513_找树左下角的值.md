## 二、思路

* 找最左下角的值，前，中，后序无所谓本次不需要中间节点做什么，所以一定是左侧最为优先。最左侧，既找到叶子节点。
* 递归三部曲
* **确认返回值和参数类型**：不需要处理返回值，具体处理当前节点和当前节点深度的变量。

```cpp
void traversal(TreeNode* cur, int depth);
```

* **确认终止条件**：找到各叶子节点，根据它们所处的深度，找出最深的叶子节点的数据。

```cpp
if (NULL == cur->left && NULL == cur->right){
    if (depth > maxDepth){
        result = cur->val;
    }
    return;
}
```

* **单次递归实现的逻辑**：判断当前节点是否是最左叶子节点，若是将当前深度比最大深度大，更新全局结果变量。若不是，遍历当前节点的左右节点，并将传入的深度+1

```cpp
if (NULL == cur->left && NULL == cur->right){
    if (depth > maxDepth){
        result = cur->val;
    }
    return;
}

if (NULL != cur->left){
    traversal(cur-left, depth+1);		// 回溯的思想
}

if (NULL != cur->right){
    traversal(cur->right, depth + 1);
}

return;
```

## 三、代码

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    int result = 0;
    int maxDepth = INT_MIN;

    void traversal(TreeNode* cur, int depth){
        // 若是遇到叶子节点，与最大深度比较，若大于更新结果。
        if (NULL == cur->left && NULL == cur->right){
            if (depth > maxDepth){
                maxDepth = depth;
                result = cur->val;
            }
            return ;
        }

        if (NULL != cur->left){
            traversal(cur->left, depth + 1);
        }

        if (NULL != cur->right){
            traversal(cur->right, depth + 1);
        }

    }

    int findBottomLeftValue(TreeNode* root) {
        if (NULL != root){
            traversal(root, 0);
        }
        return result;
    }
};
```
