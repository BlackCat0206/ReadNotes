# 105_从前序与中序遍历序列构造二叉树

---- Lee

## 一、题目

给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是二叉树的**先序遍历**， `inorder` 是同一棵树的**中序遍历**，请构造二叉树并返回其根节点。

**示例 1:**

![img](https://assets.leetcode.com/uploads/2021/02/19/tree.jpg)

```
输入: preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
输出: [3,9,20,null,null,15,7]
```

**示例 2:**

```
输入: preorder = [-1], inorder = [-1]
输出: [-1]
```

**提示:**

- `1 <= preorder.length <= 3000`
- `inorder.length == preorder.length`
- `-3000 <= preorder[i], inorder[i] <= 3000`
- `preorder` 和 `inorder` 均 **无重复** 元素
- `inorder` 均出现在 `preorder`
- `preorder` **保证** 为二叉树的前序遍历序列
- `inorder` **保证** 为二叉树的中序遍历序列

## 二、思路

迭代，切割

* 前序的第一个节点为通过`find`和`distance`算法，查找到对应元素的迭代器，并计算在中序遍历的距离，切割。
* 注意：前序遍历不能pop首元素，故在构造时套对`preoerder.begin() + 1`
* 时间复杂度：O(n^2)
* 空间复杂度：O(n^2)

哈希：

* 预处理`inorder`每个元素的下标，这样就可以`O(1)`查到`preorder[0]`在`inorder`的位置，从而`O(1)`就知道左子树的大小。
* 把递归参数改为数组的下标区间（左闭右开区间）的左右端点，从而避免复制数组。

## 三、代码

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:

    TreeNode* dfs(vector<int>& preorder, int pre_left, int pre_right, 
    vector<int>& inorder, int in_left, int in_right, unordered_map<int, int>& index)
    {
        // 左闭右开区间，相等既返回
        if (pre_left == pre_right){
            return nullptr;
        }

        // 计算左子树的长度
        int left_size = index[preorder[pre_left]] - in_left;
        // 创建左子节点
        TreeNode* left = dfs(
            preorder, pre_left + 1, pre_left + 1 + left_size,
            inorder, in_left, in_left + left_size,
            index);
        
        // 创建右子节点
        TreeNode* right = dfs(
            preorder, pre_left + 1 + left_size, pre_right,
            inorder, in_left + left_size + 1, in_right,
            index
        );

        return new TreeNode(preorder[pre_left], left, right);
    }

    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        int n = inorder.size();
        unordered_map<int, int> index;
        for (int i = 0; i < n; ++i){
            index[inorder[i]] = i;
        }
        return dfs(preorder, 0, n, inorder, 0, n, index);
    }   
};
```
