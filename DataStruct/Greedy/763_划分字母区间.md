# 763_划分字母区间

---- Lee

## 一、题目

给你一个字符串 `s` 。我们要把这个字符串划分为尽可能多的片段，同一字母最多出现在一个片段中。例如，字符串 `"ababcc"` 能够被分为 `["abab", "cc"]`，但类似 `["aba", "bcc"]` 或 `["ab", "ab", "cc"]` 的划分是非法的。

注意，划分结果需要满足：将所有划分结果按顺序连接，得到的字符串仍然是 `s` 。

返回一个表示每个字符串片段的长度的列表。

**示例 1：**

```
输入：s = "ababcbacadefegdehijhklij"
输出：[9,7,8]
解释：
划分结果为 "ababcbaca"、"defegde"、"hijhklij" 。
每个字母最多出现在一个片段中。
像 "ababcbacadefegde", "hijhklij" 这样的划分是错误的，因为划分的片段数较少。 
```

**示例 2：**

```
输入：s = "eccbbbbdec"
输出：[10]
```

**提示：**

- `1 <= s.length <= 500`
- `s` 仅由小写英文字母组成

## 二、思路

**划分规则：**

找到遍历过字符最远出现位置，这就是划分点。

**如何统计最远出现位置？**

`Hash`：元素作为key，最远出现位置为value。

**结果如何计算？**

设置起始地址`left`，与遍历元素最远位置`right`。结果`right - left + 1`。

## 三、代码

```cpp
class Solution {
public:
    vector<int> partitionLabels(string s) {
        vector<int> ans;
        int hash[27] = {0};
        for (int i = 0; i < s.size(); ++i){
            hash[s[i] - 'a'] = i;
        }

        int left = 0;   // 分割字符串起点
        int right = 0;  // 分割字符串终点
        for (int i = 0; i < s.size(); i++){
            right = max(hash[s[i] - 'a'], right); // 计算遍历过元素的最远位置
            if (i == right){
                ans.push_back(right - left + 1);
                left = i + 1;
            }
        }
        return ans;
    }
};
```
