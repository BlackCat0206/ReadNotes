## 一、题目

给定一个二叉树，返回所有从根节点到叶子节点的路径。

说明: 叶子节点是指没有子节点的节点。

## 二、思路

* 前序遍历以及回溯的过程如图：

  ![257.二叉树的所有路径](https://file1.kamacoder.com/i/algo/20210204151702443.png)

  我们先使用递归的方式，来做前序遍历。**要知道递归和回溯就是一家的，本题也需要回溯。**

* 递归三要素

* 确认函数返回值类型和参数：不需要处理返回值设置为`void`，处理当前节点向叶子节点的路径，传入当前节点，传入一个`vector<int> vec`用于存储到当前节点之前的节点内容，`vector<string> path`当遇到叶子节点时，将所有数值转换为`string`类型作为结果存储。

```cpp
void findPath(TreeNode* cur, vector<int>& nums, vector<string>& path)
```

* 终止条件：遇到叶子节点即可，既当前节点的左右子节点均为空，

```cpp
if (NULL == cur->left && NULL == cur->right){
    处理逻辑
}
```

* 单次递归要处理的逻辑：

## 三、代码

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    void findPath(TreeNode* cur, vector<int>& pathNums, vector<string>& path){
        pathNums.push_back(cur->val);                           // 先填充中间节点的数值
        if (NULL == cur->left && NULL == cur->right){
            // 将存储的数据转换为字符串并记录
            string spath;
            for (int i = 0; i < pathNums.size() - 1; i++){
                spath += to_string(pathNums[i]);
                spath += "->";
            }
            // 叶子节点之后不需要添加"->"
            spath += to_string(pathNums[pathNums.size() - 1]);
            path.push_back(spath);
        }

        // 遍历左子树
        if (NULL != cur->left){
            findPath(cur->left, pathNums, path);
            pathNums.pop_back();                // 回溯
        }

        // 遍历右子树
        if (NULL != cur->right){
            findPath(cur->right, pathNums, path);
            pathNums.pop_back();                // 回溯
        }
    }

    vector<string> binaryTreePaths(TreeNode* root) {
        vector<string> path;
        if (NULL == root){
            return path;
        }
        vector<int> pathNums;
        findPath(root, pathNums, path);
        return path;
    }
};
```
