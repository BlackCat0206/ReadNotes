# 559_n叉树的最大深度

---- Lee

## 一、题目

给定一个 n 叉树，找到其最大深度。

最大深度是指从根节点到最远叶子节点的最长路径上的节点总数。

例如，给定一个 3叉树 :

![559.n叉树的最大深度](https://file1.kamacoder.com/i/algo/2021020315313214.png)

我们应返回其最大深度，3。

[力扣题目链接(opens new window)](https://leetcode.cn/problems/maximum-depth-of-n-ary-tree/)

## 二、思路

* 递归三要素
* 确认函数返回值和参数：函数返回当前节点所在的最大高度，参数为当前节点。

```cpp
int maxDepth(TreeNode* cur);
```

* 确认终止条件：当节点为空时，则说明最大深度，最小高度。

```cpp
if (NULL == cur) return 0;
```

* 获取当前节点子节点的高度进行比较，取最大值。

```cpp
int depth = 0;
for (int i = 0; i < root->children.size(); i++){
    depth = max(depth, maxDepth(root->children[i]));
}
return 1 + depth;
```

## 三、代码

```cpp
/*
// Definition for a Node.
class Node {
public:
    int val;
    vector<Node*> children;

    Node() {}

    Node(int _val) {
        val = _val;
    }

    Node(int _val, vector<Node*> _children) {
        val = _val;
        children = _children;
    }
};
*/

class Solution {
public:
    int maxDepth(Node* root) {
       if (NULL == root){
          return 0;
       } 
       int depth = 0;
       for (int i = 0; i < root->children.size(); i++){
        depth = max(depth, maxDepth(root->children[i]));
       }
       return 1 + depth;
    }
};
```
