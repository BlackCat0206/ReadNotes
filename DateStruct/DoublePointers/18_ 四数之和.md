---- Lee

## 一、题目

题意：给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 d ，使得 a + b + c + d 的值与 target 相等？找出所有满足条件且不重复的四元组。

**注意：**

答案中不可以包含重复的四元组。

示例： 给定数组 nums = [1, 0, -1, 0, -2, 2]，和 target = 0。 满足要求的四元组集合为： [ [-1, 0, 0, 1], [-2, -1, 1, 2], [-2, 0, 0, 2] ]

[力扣题目链接(opens new window)](https://leetcode.cn/problems/4sum/)

## 二、思路

思路相较于之前的[15_三数之和.md](./15_三数之和.md)

* `target`时任意值，不能在排序后看到nums[i]>target就可以终止！
  * 例如{ -5，-4，-3 ，0，0}  target:-7, 若延续之前的逻辑在-4时直接终止则错误。
* 可以使用：nums[i] > target && nums[i] >= 0)

* 在“三数之和”的for循环基础之上，在加入一个for循环实现：nums[k] + nums[i] + nums[left] + nums[right];

## 三、代码

```cpp
class Solution {
public:
    vector<vector<int>> fourSum(vector<int>& nums, int target) {
        // 对数据进行排序
        sort(nums.begin(), nums.end());

        // 初始换返回值
        vector<vector<int>> result;

        // 创建操作下标
        int left = 0;
        int right = 0;

        // 建立双重循环实现双指针操作
        for (int k = 0; k < nums.size(); k++)
        {
            // 在target非负数的情况下，进行条件判断与返回
            if (nums[k] > target && nums[k] >=0)
            {
                break;
            }

            // 完成去重操作
            if (k > 0 && nums[k] == nums[k - 1])
            {
                continue;
            }

            for (int i = k + 1; i < nums.size(); i++)
            {
                // 在target非负数的情况下，进行条件判断与返回
                if ((nums[k] + nums[i] > target) && (nums[k] + nums[i] >= 0))
                {
                    break;
                }

                // 完成去重操作
                if (i > k + 1 && nums[i] == nums[i - 1])
                {
                    continue;
                }

                // 设置动态数据边界
                left = i + 1;
                right = nums.size() - 1;
                while(left < right)
                {
                    // 考虑到输入数据特大时，转变为long类型
                    if ((long)nums[k] + nums[i] + nums[left] + nums[right] > target)
                    {
                        --right;  
                    }
                    else if ((long)nums[k] + nums[i] + nums[left] + nums[right] < target)
                    {
                        ++left;
                    }
                    else
                    {
                        // 填充结果
                        result.push_back(vector<int>{nums[k], nums[i], nums[left], nums[right]});
                        // 对计算空间去重
                        while(left < right && nums[left] == nums[left + 1])
                        {
                            left++;
                        }
                        while(left < right && nums[right] == nums[right - 1])
                        {
                            right--;
                        }

                        // 收缩空间
                        left++;
                        right--;
                    }
                }
            }
        }
        return result;
    }
};
```
