# 15_三数之和

---- Lee

## 一、题目

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且**不重复的三元组**。

**注意：** 答案中不可以包含重复的三元组。

示例：

给定数组 nums = [-1, 0, 1, 2, -1, -4]，

满足要求的三元组集合为： [ [-1, 0, 1], [-1, -1, 2] ]

[力扣题目链接(opens new window)](https://leetcode.cn/problems/3sum/)

## 二、思路

* 为什么不用Hash：题目要求不能有重复的三元组。若使用hash，去重的操作会让人吐血！
* 三指针思路：
  * 对数据进行排序，从小到大。
    * 在nums[i] > 0时返回结果。排序后大于0的数与其后相加的结果依然大于0。
  * 固定一个首指针，设置left和right，进行轮询计算得到结果，进行范围收缩（参考二分法对target）。
  * 注意起始元素的去重方式规避输入{0，0，0}去重导致无结果，与{-1，-1，1，0}不去重导致有重复的元素组。

## 三、代码

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        // 先对输入的数组进行升序排序
        sort(nums.begin(), nums.end());

        vector<vector<int>> result;
        int left = 0;
        int right = nums.size() - 1;
        int sum = 0;
        for (int i = 0; i < nums.size(); ++i)
        {
            // 如果当前的起始数据已经大于0，则不会出现与其后数据相加为0的情况
            if (0 < nums[i])
            {
                break;
            }

            // 正确的去重：规避上去除述情况{-1，-1，1，0} 结果会出现重复的元素
            if (i > 0 && nums[i] == nums[i - 1])
            {
                continue;
            }

            // 错误的去除方法：会去除{-1，-1，2}的情况
            // if (nums[i] == nums[i + 1])
            // {
            //     continue;
            // }

            left = i + 1;
            right = nums.size() - 1;

            while(left < right)
            {
                sum = nums[i] + nums[left] + nums[right];

                if (sum > 0)
                {
                    --right;
                }
                else if (sum < 0)
                {
                    ++left;
                }
                else
                {
                    result.push_back(vector<int>{nums[i], nums[left], nums[right]});
                    // 在填充数据后，进行去重。
                    while(left < right && nums[left] == nums[left + 1])
                    {
                        ++left;
                    }
                    while(left < right && nums[right] == nums[right - 1])
                    {
                        --right;
                    }

                    // 收缩数据区域
                    left++;
                    right--;
                }
            }
        }
        return result;
    }
};
```
