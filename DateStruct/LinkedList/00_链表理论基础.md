# 00_链表理论基础

---- Lee

## 一、概述

链表是通过指针串联起来的线性结构，每一个节点由两个部分构成：指针域和数据域。

## 二、链表的类型

* 单向链表
* 双向链表
* 循环链表

## 三、链表的存储方式

* 在内存中存储不连续
* 通过指针域连接各个节点。

## 四、链表的定义

```c
struct ListNode
{
	int value;
    struct ListNode *node;
};

typedef struct ListNode ListNode;
```

## 五、性能分析

|      | 插入/删除（时间复杂度） | 查询（时间复杂度） | 适用场景                       |
| ---- | ----------------------- | ------------------ | ------------------------------ |
| 数组 | O(n)                    | O(1)               | 数据固定、频繁查询，较少增删   |
| 链表 | O(1)                    | O(n)               | 数据不固定、较少查询、频繁增删 |
