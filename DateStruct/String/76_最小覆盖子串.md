# 76_最小覆盖子串

---- Lee

## 一、题目

给定两个字符串 `s` 和 `t`，长度分别是 `m` 和 `n`，返回 s 中的 **最短窗口 子串**，使得该子串包含 `t` 中的每一个字符（**包括重复字符**）。如果没有这样的子串，返回空字符串 `""`。

测试用例保证答案唯一。

**示例 1：**

```
输入：s = "ADOBECODEBANC", t = "ABC"
输出："BANC"
解释：最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。
```

**示例 2：**

```
输入：s = "a", t = "a"
输出："a"
解释：整个字符串 s 是最小覆盖子串。
```

**示例 3:**

```
输入: s = "a", t = "aa"
输出: ""
解释: t 中两个字符 'a' 均应包含在 s 的子串中，
因此没有符合条件的子字符串，返回空字符串。
```

**提示：**

- `m == s.length`
- `n == t.length`
- `1 <= m, n <= 105`
- `s` 和 `t` 由英文字母组成

**进阶：**你能设计一个在 `O(m + n)` 时间内解决此问题的算法吗？

## 二、思路

* 题目要求，包含每一个字符即可，不需要类似于`strstr()`严格匹配字符串中的顺序。
* 首先可以统计字串中每个元素对应的个数：

```cpp
vector<int> charCount(128, 0);
for (char c : t){
    charCount[c]++;
}
```

* 使用滑动窗口，指针向右移动，若当前元素在子串中，`count++;`，同时对于字串的当前元素数量-1（表示已经匹配了），右指针移动。

```cpp
while (right <= S.size()){
    if (charCount[S[right]] > 0){
        ++count;
    }
    --charCount[S[right]];
    ++right;
}
```

* 当完全匹配后，循环收缩左侧窗口：
  * 更新最小长度，记录当前字符串的起始位置。
  * 子串计数中左侧元素+1，判断其是否>0，若>0说明左侧元素确实是字串的一部分，count--；若不是，则说明不是字串一部分或者说其后有其他字串。
  * 收缩左指针。

```cpp
while (count == T.size())
{
    if (minLen > right - left){
        minLen = right - left;
        start = left;
    }
    
    ++charChount[S[left]];
    if (charCount[S[left]] > 0)
    {
        --count;
    }
    --left;
}
```

* 遍历结束，判断最小长度是否为最大值，若是返回空string，若不是返回string的substr。

```cpp
return minLen == INT_MAX ? "" : S.substr(start, minLen);
```

## 三、代码

```cpp
class Solution {
public:
    string minWindow(string s, string t) {
        if (s.empty() || t.empty()){
            return "";
        }

        int sizeS = s.size();
        int sizeT = t.size();

        vector<int> charCount(128, 0);
        for (char c : t)
        {
            ++charCount[c];
        }

        int left = 0;
        int right = 0;
        int start = 0;
        int count = 0;
        int minLen = INT_MAX;
        
        while (right <= sizeS)
        {
            if (charCount[s[right]] > 0)
            {
                ++count;
            }
            --charCount[s[right]];
            ++right;

            while (sizeT == count)
            {
                if (minLen > right - left)
                {
                    minLen = right - left;
                    start = left;
                }

                ++charCount[s[left]];
                if (charCount[s[left]] > 0)
                {
                    --count;
                }
                ++left;
            }
        }

        return minLen == INT_MAX ? "" : s.substr(start, minLen);
    }
};
```
