# 17_14_最小K个值

---- Lee

## 一、题目

设计一个算法，找出数组中最小的k个数。以任意顺序返回这k个数均可。

**示例：**

```
输入： arr = [1,3,5,7,2,4,6,8], k = 4
输出： [1,2,3,4]
```

**提示：**

- `0 <= len(arr) <= 100000`
- `0 <= k <= min(100000, len(arr))`

## 二、思路

* 堆排序
  * 大堆顶，堆顶是当前堆中的最大值。
  * 先传入k个数据。
  * 将后序数据于当前堆顶比较，若`value < 堆顶`，`pop`堆顶的元素，添加当前的元素。
* 参数判断：
  * 入参`arr`为空，`k`>`arr`数组长度，`k==0`

## 三、代码

```cpp
class Solution {
public:
    vector<int> smallestK(vector<int>& arr, int k) {
        vector<int> res;
        if (0 == arr.size() || k > arr.size() || k <= 0)
        {
            return res;
        }

        // 使用大堆顶，排序
        priority_queue<int, vector<int>> pq;
        for (auto n : arr)
        {
            // 先填充大堆顶
            if (k > pq.size())
            {
                pq.push(n);
            }
            else if (n < pq.top())
            {
                pq.pop();
                pq.push(n);
            }
        }

        while(!pq.empty())
        {
            res.push_back(pq.top());
            pq.pop();
        }
        return res;

    }
};
```
