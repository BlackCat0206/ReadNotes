# 349_两个数组的交集

---- Lee

## 一、题目

给定两个数组 `nums1` 和 `nums2` ，返回 *它们的 交集* 。输出结果中的每个元素一定是 **唯一** 的。我们可以 **不考虑输出结果的顺序** 。

**示例 1：**

```
输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2]
```

**示例 2：**

```
输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出：[9,4]
解释：[4,9] 也是可通过的
```

`LeetCode`链接：[349.两个数组的交集](https://leetcode.cn/problems/intersection-of-two-arrays/description/)

## 二、思路

* 去重：根据示例中给出的结果，其是去重的。对于去重，可以选择`set`相关容器。
* 是否需要快速查找？：`set`实现底层位红黑树、查询时间O(log n)，`unoredered_set`查询时间为：O(1)。所以我们选择`unoredered_set`。

## 三、代码

```cpp
class Solution {
public:
    vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        
        // 填充交集的数值，并进行去重
        unordered_set<int> result_nums;
        
        // 对数组1进行去重
        unordered_set<int> set1(nums1.begin(), nums1.end());

        // 遍历数组2中的元素进行查找，将找到的内容填充进result_nums，其可以去重
        for (int &num : nums2)
        {
            if (set1.find(num) != set1.end())
            {
                result_nums.insert(num);
            }
        }

        // 将去重的结果转换为vector<int>
        vector<int> vecResult(result_nums.begin(), result_nums.end());

        return vecResult;
    }
};
```
