# 454.四数相加II

## 一、题目

给定四个包含整数的数组列表 A , B , C , D ,计算有多少个元组 (i, j, k, l) ，使得 A[i] + B[j] + C[k] + D[l] = 0。

为了使问题简单化，所有的 A, B, C, D 具有相同的长度 N，且 0 ≤ N ≤ 500 。所有整数的范围在 -2^28 到 2^28 - 1 之间，最终结果不会超过 2^31 - 1 。

**例如:**

输入:

- A = [ 1, 2]
- B = [-2,-1]
- C = [-1, 2]
- D = [ 0, 2]

输出:

2

**解释:**

两个元组如下:

1. (0, 0, 0, 1) -> A[0] + B[0] + C[0] + D[1] = 1 + (-2) + (-1) + 2 = 0
2. (1, 1, 0, 0) -> A[1] + B[1] + C[0] + D[0] = 2 + (-1) + (-1) + 0 = 0

[力扣题目链接(opens new window)](https://leetcode.cn/problems/4sum-ii/)

## 二、思路

* 只计算可以实现相加为0的组合**个数**，不考虑组合中的元素是否是重复的。故将a，b的和作为key值，将其对应value++，将C、D的和取反查询是否存在，若存在将其出现次数进行累加。

* 为了方便后续的快速查询，使用`unordered_map<int, int>`

## 三、代码

```cpp
class Solution {
public:
    int fourSumCount(vector<int>& nums1, vector<int>& nums2, vector<int>& nums3, vector<int>& nums4) {
        // 将数组num1,num2和与出现次数进行统计
        unordered_map<int, int> map;
        for (int a : nums1)
        {
            for (int b : nums2)
            {
                map[a + b]++;
            }
        }

        int count = 0;
        // 遍历nums3,nums4
        for (int c : nums3)
        {
            for (int d : nums4)
            {
                if (map.find(0 - (c + d)) != map.end())
                {
                    count += map[0 - (c + d)];
                }
            }
        }
        return count;
    }   
};
```

- 时间复杂度: O(n^2)
- 空间复杂度: O(n^2)，最坏情况下A和B的值各不相同，相加产生的数字个数为 n^2
