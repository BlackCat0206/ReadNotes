# C运算符优先级表格

***---- Li Jinsheng***

***



| 优先级 | 运算符                                                       | 结合律   |
| ------ | ------------------------------------------------------------ | -------- |
| 1      | （）、[]、.、->                                              | 从左向右 |
| 2      | -、+（一元）、--、++、`sizeof`、&（取地址）、！、*（解引用）、（type）、~（自反） | 从右向左 |
| 3      | *、/、%                                                      | 从左向右 |
| 4      | -   + （二元）                                               | 从左向右 |
| 5      | << 左移、>> 右移                                             | 从左向右 |
| 6      | <、>、 >=、 <=                                               | 从左向右 |
| 7      | == !=                                                        | 从左向右 |
| 8      | & （按位与）                                                 | 从左向右 |
| 9      | ^ （按位异或）                                               | 从左向右 |
| 10     | \|（按位或）                                                 | 从左向右 |
| 11     | &&                                                           | 从左向右 |
| 12     | \|\|                                                         | 从左向右 |
| 13     | ?:                                                           | 从右向左 |
| 14     | =、/=、*=、%=、+=、-=、<<=、>>=、&=、^=、\|=                 | 从右向左 |
| 15     | ?=                                                           | 从右向左 |
| 16     | ，                                                           | 从左向右 |

**简单记**：！> 算数运算符 > 位运算符（左移、右移） > 关系运算符 > 位运算符（与、异或、或）> 逻辑运算符 > 赋值运算符 > ,(逗号)。
