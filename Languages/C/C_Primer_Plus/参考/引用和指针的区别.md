# 引用和指针的区别

***---- Li Jinsheng***

***

* **初始化**：
  * 引用：必须初始化。
  * 指针：可以只声明不初始化。
* **语法表现不同**：

```cpp
int i = 0;
int &ci = i;	// 引用
int *pi = &i;	// 指针
```

* **内存空间**：
  * 引用：引用在底层实现上**可能使用指针方式**，但**语法**上**不占用额外内存**。
  * 指针：存储所指对象的地址，有自己的内存空间。
* **空值可能性**：
  * 引用：不能为空。
  * 指针：可以置为空（`NULL` 或 `nullptr`）。
* **可修改性**：
  * 引用：初始化完成，引用和其对象就绑定在一起，不能修改。
  * 指针：可以修改指向。
