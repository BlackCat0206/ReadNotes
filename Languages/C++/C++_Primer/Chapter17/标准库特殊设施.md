## 二、内容

### 2.1 tuple 类型

`tuple`是一个模板类型，可将不同类型的多个值打包成一个单一对象，类似 “快速搭建的结构体”，无需定义类即可聚合多个数据成员。

#### 2.1.1 定义和初始化 tuple

```cpp
#include <tuple>
// 初始化：类型与值一一对应，数量不限
std::tuple<std::string, int, double> t1("Tom", 20, 98.5);
// 默认初始化：成员均默认初始化
std::tuple<int, std::string, bool> t2;
// make_tuple：根据实参推导类型
auto t3 = std::make_tuple(10, "Lily", 3.14);
```

#### 2.1.2 访问 tuple 成员

通过`std::get<索引>`访问，索引从 0 开始，且必须是编译期常量；`tuple_size`获取成员数量，`tuple_element<索引, 类型>::type`获取指定位置成员的类型。

```cpp
// 访问t1的第一个成员（string类型）
std::string name = std::get<0>(t1);
// 修改第二个成员（int类型）
std::get<1>(t1) = 21;

// 获取tuple成员数量
using TupType = decltype(t1);
size_t sz = std::tuple_size<TupType>::value; // sz = 3
// 获取第3个成员的类型
using ElemType = std::tuple_element<2, TupType>::type; // ElemType = double
```

#### 2.1.3 tuple 的比较

当 tuple 中所有成员都支持比较操作时，tuple 可进行 ==、!=、<、> 等比较，比较规则：按成员顺序逐一比较，第一个不相等的成员决定整体比较结果。

```cpp
std::tuple<int, string> t4(1, "a"), t5(2, "b");
bool res1 = (t4 < t5); // true，第一个成员1 < 2
std::tuple<int, string> t6(1, "c"), t7(1, "b");
bool res2 = (t6 < t7); // false，第一个成员相等，第二个"c" > "b"
```

#### 2.1.4 tuple 的应用场景

常用于函数返回多个不同类型的值，无需自定义结构体，简化代码。

```cpp
std::tuple<std::string, int, double> getStudentInfo() {
    return {"Jerry", 18, 95.0};
}
```

### 2.2 bitset 类型

`bitset`用于处理固定长度的二进制位序列，提供便捷的位操作接口，相比内置位运算更易读、易维护。

#### 2.2.1 定义和初始化 bitset

```cpp
#include <bitset>
// 初始化：指定长度为8，所有位初始化为0
std::bitset<8> b1;
// 用无符号整数初始化：0b1010 → 十进制10
std::bitset<8> b2(10); // 00001010
// 用字符串初始化（仅含0/1），高位对应字符串左侧
std::bitset<8> b3("1010"); // 00001010
std::bitset<8> b4("11001010"); // 11001010
```

#### 2.2.2 bitset 的位操作

支持位运算（&、|、^、~）、移位（<<、>>）、置位 / 复位 / 翻转等操作，还可访问指定位。

```cpp
// 访问第3位（从0开始）
bool bit3 = b4[3]; // b4是11001010，第3位是1
// 置位：将第1位设为1
b4.set(1); // 11001010 → 11001011
// 复位：将第7位设为0
b4.reset(7); // 11001011 → 01001011
// 翻转所有位
b4.flip(); // 01001011 → 10110100
// 位运算
std::bitset<8> b5 = b2 | b4; // 按位或
```

#### 2.2.3 bitset 的查询操作

`count()`统计置位（1）的位数，`size()`获取总位数，`test(位索引)`检查指定位是否为 1，`any()`判断是否有位为 1，`none()`判断是否所有位为 0。

```cpp
size_t cnt = b4.count(); // 统计1的位数
bool hasOne = b4.any(); // 有1则返回true
bool allZero = b4.none(); // 全0则返回true
bool bit5 = b4.test(5); // 检查第5位是否为1
```

### 2.3 正则表达式

C++11 引入`<regex>`库，支持正则表达式匹配、查找、替换等操作，核心组件包括`regex`（正则表达式对象）、`smatch`（存储字符串匹配结果）、`regex_match`/`regex_search`/`regex_replace`（操作函数）。

#### 2.3.1 基本匹配（regex_match）

匹配整个字符串是否符合正则表达式，完全匹配才返回 true。

```cpp
#include <regex>
#include <string>
#include <iostream>

std::string str = "123456";
// 正则：匹配1-9开头的6位数字
std::regex reg(R"(\d{6})"); 
bool isMatch = std::regex_match(str, reg); // true
```

#### 2.3.2 查找匹配（regex_search）

查找字符串中第一个匹配正则的子串，无需完全匹配。

```cpp
std::string str2 = "tel:123456, email:abc@163.com";
std::regex reg2(R"(\d{6})");
std::smatch result;
if (std::regex_search(str2, result, reg2)) {
    // result[0]存储匹配到的整个子串
    std::cout << "匹配到：" << result.str() << std::endl; // 输出123456
    // result.position()返回匹配子串的起始位置
    std::cout << "起始位置：" << result.position() << std::endl;
}
```

#### 2.3.3 正则替换（regex_replace）

替换字符串中匹配正则的子串。

```cpp
std::string str3 = "hello 123 world 456";
std::regex reg3(R"(\d+)");
// 将所有数字替换为[number]
std::string newStr = std::regex_replace(str3, reg3, "[number]");
// newStr = "hello [number] world [number]"
```

#### 2.3.4 正则表达式语法

支持标准正则语法，如`\d`（数字）、`\w`（字母 / 数字 / 下划线）、`+`（1 次或多次）、`*`（0 次或多次）、`{n}`（n 次）、`[]`（字符集）等；使用原始字符串（R"()"）避免转义符冲突。

### 2.4 随机数

C++11 摒弃旧的`rand()`，提供`<random>`库，包含随机数引擎（生成随机数序列）和分布（将引擎输出映射到指定范围 / 分布），保证随机数的质量和可控性。

#### 2.4.1 随机数引擎和分布

```cpp
#include <random>

// 随机数引擎：mt19937（梅森旋转算法，高效且周期长）
std::mt19937 engine(std::random_device{}()); 
// 均匀分布：生成1-100的整数
std::uniform_int_distribution<int> dist(1, 100);
// 生成随机数
int randNum = dist(engine); // 每次调用生成1-100的随机整数
```

#### 2.4.2 常用分布类型

- 均匀分布：`uniform_int_distribution`（整数）、`uniform_real_distribution`（浮点数）
- 正态分布：`normal_distribution`（指定均值和标准差）
- 伯努利分布：`bernoulli_distribution`（bool 值，指定 true 的概率）

```cpp
// 正态分布：均值0，标准差1
std::normal_distribution<double> normDist(0, 1);
double normNum = normDist(engine);

// 伯努利分布：true概率0.5
std::bernoulli_distribution bernDist(0.5);
bool bernRes = bernDist(engine);
```

#### 2.4.3 随机数引擎的特性

- 引擎对象是状态的，每次调用会更新状态，生成下一个随机数；
- 若需可复现的随机序列，可固定引擎的种子（如`mt19937 engine(12345);`）；
- `random_device`用于生成真随机种子（部分平台可能模拟）。

### 2.5 IO 库的高级特性

#### 2.5.1 格式化输入输出

- 控制整数进制：`hex`（十六进制）、`oct`（八进制）、`dec`（十进制），需包含`<iomanip>`；
- 控制浮点数精度：`setprecision`（指定有效数字 / 小数位数）；
- 控制输出宽度和填充：`setw`（设置下一个输出的宽度）、`setfill`（填充字符）。

```cpp
#include <iomanip>
#include <iostream>

int num = 255;
std::cout << std::hex << num << std::endl; // 输出ff（十六进制）
double pi = 3.1415926;
// 保留3位小数（fixed表示固定小数位）
std::cout << std::fixed << std::setprecision(3) << pi << std::endl; // 3.142
// 设置输出宽度为8，填充字符为*
std::cout << std::setw(8) << std::setfill('*') << num << std::endl; // ***   255
```

#### 2.5.2 未格式化的 IO 操作

直接操作字符 / 字节，如`get()`（读取单个字符，包括空白符）、`put()`（写入单个字符）、`read()`（读取指定字节到缓冲区）、`write()`（写入缓冲区内容）。

```cpp
// 读取单个字符
char c;
std::cin.get(c); // 读取包括空格、换行的字符
// 写入单个字符
std::cout.put('A');

// 读取字节到缓冲区
char buf[1024];
std::cin.read(buf, 10); // 最多读取10字节到buf
// 输出读取的字节数
std::cout << std::cin.gcount() << std::endl;
// 写入缓冲区内容
std::cout.write(buf, std::cin.gcount());
```

#### 2.5.3 流的随机访问

- `seekg`（输入流）/`seekp`（输出流）：移动流的读写位置；
- `tellg`（输入流）/`tellp`（输出流）：获取当前读写位置；
- 位置类型：`streampos`（字节位置）、`streamoff`（偏移量）。

```cpp
#include <fstream>

std::fstream fs("test.txt", std::ios::in | std::ios::out);
// 移动到文件末尾（偏移量0，参照点end）
fs.seekg(0, std::ios::end);
// 获取文件大小
std::streampos fileSize = fs.tellg();
// 移动到文件开头
fs.seekg(0, std::ios::beg);
```
